如何用 hugo 搭建个人博客

March 12, 2020

今天写的第一篇博客是关于“如何用 hugo 搭建个人博客”：

  1. HUGO的快速设置八部曲
  2. GitHub的新建username.github.io仓库和上传public文件夹

Part 1 HUGO的搭建 （下面例子以Windows为例）

  1. HUGOv0.67.0版本，下载对应自己电脑操作平台的压缩包，解压后得到hugo.exe。

  2. HUGO添加环境变量并验证安装

  • 把HUGO添加到环境变量中,添加成功后命令行输入hugo version 验证HUGO是否安装成功，如 Hugo Static Site Generator v0.67.0-7F1DA3EF windows/amd64 BuildDate: 2020-03-09T20:35:55Z 恭喜你，可以进入下一步操作了。

  1. 创建username.github.io-creator目录

  • hugo new site username.github.io-creator（username为你的GITHUB用户名），生成目录文件夹后，cd 文件名进入文件目录。

  1. 创建git本地仓库并添加主题

  • git init初始化一个空的git本地仓库，然后输入 git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 回车执行再次输入 echo 'theme = "ananke"' >> config.toml完成添加主题的操作。

  1. 创建.md文本并添加内容

  • hugo new posts/自己取的文章名.md ； 然后命令行输入 code . 用VS Code打开新建的文本编辑内容。 PS：需要注意的是，当你需要发布内容的时候请把 draft:改为false

  1. 开启HUGO的服务器

  • VS code终端输入 hugo server -D 会生成一个http://localhost:1313/的连接，点击既能查看到当前在本地服务所生成的个人博客。

  1. 修改config.toml

  • 如图修改languageCode，title改为自己的标题
  • example.org，替换成你自己的域名

  1. 生成静态页面

  • hugo -D 生成了一个名为public的文件夹

恭喜你！已经完成了全部HUGO的配置，接下来只需要学习如何上传到GITHUB就能完成一个属于你自己的个人博客啦~

Part 2 如何上传到GitHub##

  1. 在文件夹根目录创建.gitignore文件

  • 创建.gitignore文件，并在里面写上/public/

  1. add & commit 依次输入一下命令

  • cd public
  • git init
  • git add
  • git commit

  1. 创建仓库 & 上传public

  • 进入GitHub创建一个名为username.github.io的仓库，切记仓库名必须保证为username.github.io的形式
  • 在VS Code的终端输入仓库SSH的地址命令把public文件夹上传到username.github.io的仓库中

  1. 把public文件夹上传到GitHub创建一个名为username.github.io的仓库

  • 刷新GitHub界面可以看到``public`文件夹已经成功上传到仓库当中
  • 点击界面Settings按钮看到如下图的一样就代表你的博客已经搭建成功了，如果没有出现绿条，而是一个按钮，请选择master branch然后点击即可。

到此，我们已经完成了所有操作了！！！最后让我们来看下我们努力后的结果吧（撒花end）
